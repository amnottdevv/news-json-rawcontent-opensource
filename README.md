# 🇮🇩 Indonesia Tech News JSON (Open Source)

> 🚀 Free Indonesian News Dataset in JSON Format — No Backend, Ready to Use

Dataset berita Indonesia dalam format JSON yang dapat digunakan secara bebas untuk berbagai kebutuhan seperti web development, aplikasi mobile, API testing, hingga project portfolio.

---

# 📌 About

Repository ini menyediakan kumpulan berita Indonesia dalam format JSON yang ringan, terstruktur, dan mudah digunakan.

JSON (JavaScript Object Notation) merupakan format data yang umum digunakan dalam pengembangan web karena mudah dibaca oleh manusia dan mesin.

Project ini dibuat untuk membantu developer mendapatkan data berita tanpa harus membuat backend sendiri.

# 📂 Struktur Data

Contoh format JSON:

```json
{
  "news": [
    {
      "id": 1,
      "name": "Judul Berita",
      "artikel": "Isi lengkap berita...",
      "posted": "2026-04-01",
      "descriptions": "Deskripsi singkat berita",
      "image": "https://example.com/image.jpg",
      "topics": ["tech", "cyber"]
    }
  ]
}
```

---

# 🌐 Cara Menggunakan

Kamu bisa langsung fetch JSON dari GitHub Raw:

```javascript
fetch("https://raw.githubusercontent.com/amnottdevv/news-json-rawcontent-opensource/main/indotechnews/news1.json")
  .then(res => res.json())
  .then(data => console.log(data));
```
